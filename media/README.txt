This folder contains third-party browser libraries shipped with the extension for use in the webview.

- mermaid.min.js (UMD) from https://cdn.jsdelivr.net/npm/mermaid@10/dist/mermaid.min.js
- svg-pan-zoom.min.js from https://cdn.jsdelivr.net/npm/svg-pan-zoom@3.6.1/dist/svg-pan-zoom.min.js

If you update versions, please replace the files and verify the webview works offline.
